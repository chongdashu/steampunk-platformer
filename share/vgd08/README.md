# VGD08 Prompt Templates

This folder contains the reusable public prompt templates from the Robin Chute
build.

## Included

- `src/` — Phaser source for the playable game, character playground, and
  character gym.
- `public/assets/` — runtime assets required by the included scenes.
- `share/vgd08/prompt-templates/` — public prompt templates for the key
  development beats.

## Not Included

Raw recording material, generation runs, detailed prompt captures, plans,
learnings, and heavier development tools are intentionally excluded here.

## Start Here

- [Watch the full tutorial](https://www.youtube.com/watch?v=x_P855cmBxQ&utm_source=github&utm_campaign=vgd08&utm_medium=readme)
- [Open the Excalidraw cheatsheet](https://link.excalidraw.com/readonly/OG0IHNny1uPpfEs8qMJG?utm_source=github&utm_campaign=vgd08&utm_medium=readme)

```bash
npm install
npm run dev
```

Then open `share/vgd08/prompt-templates/README.md` for the reusable prompts.
